# 0x01 Dependences 依赖
> * Python3
> * pyquery
> * pycurl
> * BytesIO
> * sqlite3
> * itertools
> * threading

# 0x02 How to use 使用方法
```bash
\> python3 main.py
\> 输入本次订餐地区列表页网址(例: http://waimai.meituan.com/home/wqj6x9t9310r):http://waimai.meituan.com/home/wqj6x9t9310r
\> 输入本次订餐预算:20
\> 是否使用红包? (y/n):y
\> 红包金额:1
\> 红包的最低使用金额:20
```
